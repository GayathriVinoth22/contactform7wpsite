Automatically include visitor's tracking information in Contact Form 7 Emails. 

== Description ==

This plugin is an extension to the popular "Contact Form 7" plugin. It allows website owners to get more information about the visitors who are submitting contact forms on their website. Several tracking information (like HTTP Referer) can be tracked and automatically (or manually) included in admin emails.

1. Upload the 'Referer-Tracker-For-Contact-Form-7' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
